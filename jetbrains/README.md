# Intrigued Theme for JetBrains

This directory contains the JetBrains equivalent of the Intrigued Theme originally designed for Visual Studio Code.

## Installation

1. Open your JetBrains IDE.
2. Go to `Settings` / `Preferences` -> `Editor` -> `Color Scheme`.
3. Click the gear icon next to the "Scheme:" dropdown.
4. Select `Import Scheme` -> `IntelliJ IDEA color scheme (.icls)`.
5. Select the `Intrigued.icls` file located in the `src/` directory.
6. Apply and enjoy!
